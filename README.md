vkg
===

The package manager for Vim
---------------------------

This is vkg (pronounced vackage), the package manager for Vim.  Currently *vkg* relies on the data from the 
folks at [Vimawesome](http://vimawesome.com).

Installation
------------
```
npm install -g vkg
```

Usage
-----
```
vkg search gundo
vkg install gundo
```
