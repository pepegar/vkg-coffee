vkg
===

The package manager for Vim
---------------------------

This is vkg, the package manager for Vim.  Currently *vkg* relies on the data from the 
folks at [Vimawesome][vimawesome].

Installation
------------
```
npm install vkg
```

Usage
-----
```
vkg search gundo
vkg install gundo
```
